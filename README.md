# VigiEmpleo.com

**VigiEmpleo.com** es una bolsa de empleo especializada en conectar a profesionales del sector de vigilancia y seguridad privada, como vigilantes, escoltas, operadores de medios tecnológicos (OMT), supervisores y operadores caninos, con empleadores en busca de talento capacitado. Este sistema facilita la publicación y búsqueda de ofertas laborales específicas para este sector, brindando una plataforma exclusiva para conectar talento y oportunidades en el área de seguridad privada.

## 🚀 Características

- **Registro especializado** de usuarios como empleadores o candidatos del sector de vigilancia y seguridad privada.
- **Publicación y gestión de ofertas laborales** dirigidas a profesionales en seguridad, como vigilantes, escoltas, y otros.
- **Búsqueda de empleo** avanzada y filtrada para perfiles de seguridad, incluyendo categorías y certificaciones.
- **Notificaciones por correo electrónico** para candidatos y empleadores sobre nuevas oportunidades o aplicaciones.
- **Interfaz interactiva y dinámica** con notificaciones visuales y alertas de éxito/error.
- **Seguridad en manejo de datos** mediante PHP y MySQL.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML, CSS, JavaScript, jQuery
- **Backend**: PHP
- **Base de Datos**: MySQL
- **Correo Electrónico**: PHPMailer
- **Alertas y validaciones**: SweetAlert (Swal)
- **Plantillas de correo**: TinyButStrong (TBS)

## 🚀 Uso

1. Regístrate como empleador o candidato dentro del sector de seguridad privada.
2. Los empleadores pueden publicar y gestionar ofertas laborales, seleccionando las categorías y perfiles que necesitan.
3. Los candidatos pueden buscar ofertas laborales específicas para su perfil (vigilante, escolta, etc.) y postularse.
4. Los usuarios reciben actualizaciones y notificaciones vía correo electrónico sobre oportunidades de interés.

## 🎯 Público Objetivo

Este sistema está diseñado exclusivamente para el sector de vigilancia y seguridad privada, conectando a empleadores y candidatos especializados en áreas como:

- Vigilantes
- Escoltas
- Operadores de Medios Tecnológicos (OMT)
- Supervisores
- Operadores Caninos
